# Guess the Number

Guess the Number is a simple command-line game where players try to guess a randomly generated number within a specified range. The game offers two modes: two-player mode and single-player mode against an AI opponent.

## How to Play

1. Make sure you have Python installed on your computer.
2. Clone this repository to your local machine using `git clone https://github.com/iVGeek/guess-the-number.git`.
3. Navigate to the project directory: `cd guess-the-number`.
4. Run the game by executing the Python script:

 python guess_the_number.py

5. The game will present you with a menu:
   - Choose "Two-Player Mode" to play against another person.
   - Choose "Single-Player Mode against AI" to play against the computer.
6. Follow the on-screen instructions to play the game.
   - In two-player mode, players take turns guessing the number.
   - In single-player mode, you will be competing against the AI.
7. Guess a number within the specified range when prompted.
8. The game will provide feedback on whether your guess is too high or too low.
9. Keep guessing until you correctly guess the number or run out of attempts.
10. The game will display the final result and give you the option to play again or quit.

## Game Levels

The game offers different levels of difficulty:
- Easy: Guess a number between 1 and 10, with 6 attempts.
- Medium: Guess a number between 1 and 50, with 4 attempts.
- Hard: Guess a number between 1 and 100, with 3 attempts.

When playing the game, you can choose the desired level to adjust the difficulty.
