# Guess the Number

Guess the Number is a simple command-line game where players try to guess a randomly generated number within a specified range.

## How to Play

1. Make sure you have Python installed on your computer.
2. Clone this repository to your local machine using `git clone https://github.com/your-username/guess-the-number.git`.
3. Navigate to the project directory: `cd guess-the-number`.
4. Run the game by executing the Python script: `python guess_the_number.py`.
5. Enter your name when prompted.
6. Choose a difficulty level from the menu presented.
7. Enter your guesses and receive feedback on whether your guess is too high, too low, or correct.
8. Keep guessing until you correctly guess the number or run out of attempts.
9. The game will display your final score and give you the option to play again or quit.

- Python 3.x
